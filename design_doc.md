开发个人导航站（局域网部署的），以下是主要的功能和几点要求
# 主要功能
1. 不需要任何的登录鉴权，只有在页面的右下角有一个编辑模式和展示模式的切换按钮，点击进入 "编辑模式"（编辑的时候自动保存），再次点击进入"展示模式"，进入页面永远默认"展示模式"。
   * 支持主题的扩展开发和安装。预制两套主题  Liquid Glass的苹果风 和 扁平风格（生成一份主题开发文档以供开发者做主题开发学习） 
   * 支持新增卡片分组，新增分组的时候，弹窗填写名字即可，直接在编辑模式下界面
   * 新增卡片时字段设计应该有 网址填写，获取图标（从地址 html自带的 icon 获取）或者上传图标或使用首字符加选择背景色的方式进行展示。在哪个分组下新增的直接归属到哪个分组即可，打开方式选择 是当前标签页打开还是新增标签页打开，填写标题
   * 卡片分组的维护应该支持拖动，放开即保存
   * 支持选择主题并切换，支持自定义背景图片，支持自定义背景图片和卡片的透明度和毛玻璃程度
   * 支持天气效果，预设雨天模式和 雨天模式打开会在，毛玻璃的特效上叠加出雨滴顺着剥离滑落的效果
   * 支持ctrl + k 搜索 （自动根据标题或者网址匹配）
# 主题说明
   *  Liquid Glass 的苹果风 和 扁平风格的卡片都是圆角，并且icon在左 标题在右
   *  Liquid Glass 的苹果风要求极致
# 框架的选择
   * 前端框架根据我提的ui效果和要实现的功能选择易维护轻量的方案把，视觉效果要好 
   * 后端选择你最擅长的语言开发吧 
   * 要求前后端一起不需要分离，所有的存储使用 SQLite 存储

# 部署运维
   * 写好 docker file 和 docker compose file，将数据挂载至运行目录，使用名为 wx 的外部网络
